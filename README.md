# CEO Manager Employee


## Getting Started

### Installing

Cloning the project.

```bash
git clone https://github.com/pshashanka/ceo-manager-employee
```

Access the project directory.

```bash
cd ceo-manager-employee
```

Install dependencies.

```
yarn install
```

Serve with hot reload at http://localhost:3000.

```
yarn start
```

### Testing

```
yarn test
```
